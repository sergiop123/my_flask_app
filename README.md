# Superstore Analysis with Gemini AI

## Overview
This project is a Flask-based dashboard for analyzing Superstore sales data, enhanced with an LLM-powered AI agent using **Google Gemini API**. After running any sales query, the app automatically generates **2–3 actionable business recommendations**.

## Features
- Data filters: Category, Sub-Category, Region, Segment  
- Queries: Total Sales, Total Profit, Avg Discount, Sales by Year, Profit by Year  
- Gemini AI recommendations  
- Clean UI with CSS  
- `.env` for secure API key
